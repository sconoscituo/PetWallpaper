# PetWallpaper

모든 앱 위에서 동물이 자유롭게 돌아다니는 오버레이 방식 Flutter 앱입니다.

## 개요

화면 최상위(오버레이)에 귀여운 픽셀아트 동물이 항상 표시되며, 어떤 앱을 사용하는 중에도 동물이 화면을 돌아다닙니다.

## 주요 기능

- **화면 오버레이**: 모든 앱 위에서 동물이 자유롭게 돌아다님
- **랜덤 이동 패턴**: 걷기 / 앉기 / 점프 동작 랜덤 전환
- **경계 반사**: 화면 끝에 닿으면 자동으로 방향 전환
- **터치 상호작용**: 동물을 터치하면 하트 이펙트 표시
- **드래그 이동**: 오버레이 창을 드래그해서 위치 조정 가능

## 필요 권한

| 권한 | 용도 |
|------|------|
| `SYSTEM_ALERT_WINDOW` | 다른 앱 위에 표시 (오버레이) 권한 |
| `VIBRATE` | 터치 진동 피드백 |

> Android 설정 > 앱 > PetWallpaper > 다른 앱 위에 표시 허용

## 기술 스택

- **Flutter** 3.x
- **flutter_overlay_window** ^0.4.0 — 화면 최상위 오버레이
- **provider** ^6.1.1 — 상태 관리
- **shared_preferences** ^2.2.2 — 설정 저장

## 프로젝트 구조

```
lib/
├── services/
│   └── overlay_service.dart     # 오버레이 시작/종료/권한 관리
└── features/
    └── pet/
        └── overlay_pet_widget.dart  # 동물 이동/애니메이션/터치 위젯
```

## 시작하기

```bash
flutter pub get
flutter run
```

앱 실행 후 "오버레이 시작" 버튼을 누르면 권한 요청 후 동물이 화면에 나타납니다.
